# 常见问题

## 安装问题

### 1. 安装依赖时出现错误怎么办？(针对编译运行)

- **问题**：运行 `pip install -r requirements.txt` 时出现错误
- **解决方案**：
  - 确保使用的是 Python 3.10-3.12 版本
  - 尝试使用 `pip install --upgrade pip` 更新 pip
  - 如果是网络问题，可以使用国内镜像源：
    ```bash
    pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
    ```

### 2. 应用程序无法启动怎么办？

- **问题**：双击可执行文件后应用程序无法启动
- **解决方案**：
  - 检查是否已安装所有依赖
  - 尝试以管理员身份运行
  - 查看应用程序目录下的日志文件，了解具体错误信息

## 使用问题

### 1. 生成语音时出现错误怎么办？

- **问题**：点击生成按钮后出现错误
- **解决方案**：
  - 检查网络连接（首次使用需要下载模型）
  - 确保有足够的磁盘空间
  - 尝试使用较小的文本段
  - 检查模型是否正确下载

### 2. 语音克隆效果不理想怎么办？

- **问题**：克隆的语音效果不够自然
- **解决方案**：
  - 使用更长的参考音频（15-30秒）
  - 确保参考音频清晰，无背景噪音
  - 参考说话人应保持稳定的语速和语调
  - 尝试调整克隆参数

### 3. 应用程序运行缓慢怎么办？

- **问题**：应用程序运行缓慢，生成语音时间长
- **解决方案**：
  - 确保使用 GPU 加速（如果可用）
  - 关闭其他占用系统资源的程序
  - 选择较小的模型进行测试
  - 增加系统内存（如果可能）
