# Interface Introduction

Cyrene QwenTTS GUI adopts a modern FluentUI design with a clean and intuitive interface, featuring a reasonable functional layout. This document provides a detailed introduction to the various interface components of the software.

## Main Interface Layout

The software's main interface is divided into the following main sections:

### 1. Top Navigation Bar

The side navigation bar contains the following elements:

- **Software Name**: Displays "Cyrene QwenTTS GUI"
- **Function Tabs**: 
  - Home
  - Voice Generation
  - Voice Design
  - Voice Clone
  - Audio Browser
  - Settings
  - About
- **System Buttons**: Minimize, Maximize/Restore, Close

### 2. Video Background

The software uses a video as the background.

### 3. Left Panel

The left panel contains the following functions:

- **Model Selection**: Select different Qwen-TTS models

### 4. Central Workspace

The central workspace is the core area of the software:

- **Text Input Box**: Enter text to be converted to speech
  - Supports multi-line text
- **Operation Buttons**: 
  - Generate Voice
  - Play/Pause
  - Save Audio

### 5. Bottom Status Bar

The bottom status bar displays the following information:

- **Status Message**: Shows the status of the current operation

## Functional Interface Details

### Home Interface

The home interface provides a quick overview of the software's features and functions:

- **Feature Introduction**: Brief introduction to the software's main features
- **Quick Start Guide**: Step-by-step guide for new users
- **Latest Updates**: Information about the latest version and updates

### Voice Generation Interface

This is the core interface for generating speech from text:

- **Text Input Area**: Enter text to be converted to speech
- **Model Selection**: Choose the appropriate Qwen-TTS model
- **Speaker Selection**: Select from available speakers
- **Generate Button**: Start the voice generation process
- **Preview Area**: Play and preview the generated voice

### Voice Design Interface

Used for creating voices based on text descriptions:

- **Voice Description**: Enter a detailed description of the desired voice (e.g., "A young female voice with a happy tone")
- **Text Input Area**: Enter the text to be spoken with the designed voice
- **Generate Button**: Create the voice based on the description and generate speech
- **Preview Area**: Play and preview the designed voice

### Voice Clone Interface

Used for cloning voices from reference audio:

- **Reference Audio Upload**: Upload an audio file containing the voice to be cloned
- **Text Input Area**: Enter the text to be spoken with the cloned voice
- **Clone Button**: Start the voice cloning and generation process
- **Preview Area**: Play and preview the cloned voice

### Audio Browser Interface

Used for managing and playing generated audio files:

- **File List**: Displays all generated audio files, sorted by modification time (newest first)
- **Playback Controls**: Play, pause, and adjust playback position
- **Current Playing**: Shows the currently selected audio file
- **Open Folder**: Button to open the output folder in File Explorer
- **Refresh**: Button to refresh the file list

### Settings Interface

Used to configure the software's language settings:

- **Language Selection**: Choose between English and Chinese
- **Restart Button**: Restart the software to apply language changes

### About Interface

Provides information about the software:

- **Software Information**: Version number and development information
- **Credits**: Acknowledgments and credits
- **License**: Information about the software's license

### Wizard Interface

The environment setup wizard displayed on first run:

- **Installation Type Selection**: Choose GPU version or CPU version
- **Mirror Source Selection**: Choose Aliyun mirror or official source
- **Installation Progress**: Display environment installation progress and logs

## Interface Operation Tips

### Keyboard Shortcuts

Common keyboard shortcuts:

- **Double-click**: Play audio files in the browser

### Drag and Drop Operations

- **Drag Audio Files**: Directly drag audio files to the voice cloning interface

### Right-Click Menu

Right-click in different areas to display context menus:

- **Text Input Area**: Copy, paste, clear, etc.

## Troubleshooting

### Interface Display Issues

If you encounter interface display issues, you can try the following solutions:

1. **Restart the Software**: Close and reopen the application
2. **Update Graphics Driver**: Ensure the graphics driver is up to date
3. **Run in Compatibility Mode**: Right-click the software icon → "Properties" → "Compatibility" → "Run in Compatibility Mode"

### Interface Response Slow

If the interface responds slowly, you can try the following solutions:

1. **Reduce Concurrent Tasks**: Close other programs that occupy system resources
2. **Upgrade Hardware**: If the hardware configuration is low, consider upgrading CPU or memory
