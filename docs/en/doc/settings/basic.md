# Basic Settings

This section introduces the basic settings options for Cyrene QwenTTS GUI.

## Personalization

### Language Settings

In the "Personalization" section, you can set the interface language:

- **简体中文**: Chinese interface
- **English**: English interface

### Language Switching Steps

1. In the "Settings" interface, click the "Language" dropdown menu
2. Select the language you need
3. The system will prompt "Configuration updated, effective after restart"
4. Click the "Close Program" button
5. Restart the software, and the language setting will take effect

## Notes

- Language settings require a software restart to take effect
- After restarting the software, all interface elements will switch to the selected language
- Language settings are saved in the configuration file and will be automatically applied at next startup
