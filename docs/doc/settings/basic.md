# 基本设置

本节介绍 Cyrene QwenTTS GUI 的基本设置选项。

## 个性化设置

### 语言设置

在「个性化」部分，您可以设置界面语言：

- **简体中文**: 中文界面
- **English**: 英文界面

### 语言切换步骤

1. 在「设置」界面中，点击「语言」下拉菜单
2. 选择您需要的语言
3. 系统会提示「配置已更新，重启程序后生效」
4. 点击「关闭程序」按钮
5. 重新启动软件，语言设置将生效

## 注意事项

- 语言设置需要重启软件才能生效
- 重启软件后，所有界面元素都会切换到所选语言
- 语言设置会保存在配置文件中，下次启动时会自动应用
