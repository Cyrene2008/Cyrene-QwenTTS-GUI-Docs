# 高级设置

## 环境设置

### 首次运行设置

首次运行软件时，系统会提示您选择安装类型：

- **GPU 版本**: 适合有 NVIDIA 显卡的用户，推理速度更快
- **CPU 版本**: 适合没有 GPU 的用户，兼容性更好

### 镜像源选择

安装环境时，您可以选择以下镜像源：

- **阿里云镜像**: 推荐，下载速度更快
- **官方源 (PyPI)**: 稳定可靠

温馨提示：遇到某个镜像源下载失败，可以点击重试并且切换镜像源继续安装。

## 系统要求

### 硬件要求

#### CPU版本
- **CPU**: Intel Core i5 或同等性能处理器
- **内存**: 至少 16GB RAM
- **存储空间**: 至少 12GB 可用空间

#### GPU版本（推荐）
- **CPU**: Intel Core i5 或同等性能处理器
- **内存**: 至少 16GB RAM
- **GPU**: NVIDIA 显卡，至少 8GB 显存
- **存储空间**: 至少 12GB 可用空间

### 软件要求

- **操作系统**: Windows 10/11 (64位)（仅支持）
- **Python**: 3.10-3.12
- **PyTorch**: 2.0 或更高版本
- **CUDA**: 11.7 或更高版本（用于 GPU 加速）

### 空间说明
- 单个模型约 4GB
- 环境安装约 5GB
- 因此要求至少预留 12GB 可用空间

## 故障排除

### 常见问题

1. **环境安装失败**
   - **原因**: 网络连接问题或镜像源不可用
   - **解决方法**: 尝试使用不同的镜像源，或检查网络连接；实在不行用代理（建议开启TUN模式代理）

2. **CUDA 不可用**
   - **原因**: 显卡不支持 CUDA 或驱动程序未安装，环境安装出错
   - **解决方法**: 安装最新的 NVIDIA 显卡驱动程序，或使用 CPU 版本；尝试使用整合包，已经经过开发者测试可用

3. **模型下载失败**
   - **原因**: 网络连接问题或模型服务器暂时不可用；可能是您的硬盘炸了
   - **解决方法**: 检查网络连接，稍后重试；清理空间
