# 界面介绍

Cyrene QwenTTS GUI 采用现代化的 FluentUI 设计，界面简洁直观，功能布局合理。本文档将详细介绍软件的各个界面部分。

## 主界面布局

软件主界面分为以下几个主要部分：

### 1. 顶部导航栏

侧边导航栏包含以下元素：

- **软件名称**: 显示 "Cyrene QwenTTS GUI"
- **功能选项卡**: 
  - Home
  - Voice Generation
  - Voice Design
  - Voice Clone
  - Audio Browser
  - Settings
  - About
- **系统按钮**: 最小化、最大化/还原、关闭

### 2. 视频背景

软件使用视频作为背景。

## 功能界面详解

### Home 界面

软件的欢迎界面。

### Voice Generation 界面

用于将文本转换为语音：

- **模型选择**: 选择不同的 Qwen-TTS 模型
- **说话人选择**: 选择预设的说话人
- **文本输入区**: 输入要转换的文本
- **操作按钮**: 
  - 生成音频
  - 播放/暂停
  - 另存为
- **音频播放器**: 播放生成的音频，显示播放进度和时间

### Voice Design 界面

用于根据文本描述设计声音：

- **声音描述**: 输入声音的描述（提示词）
- **文本输入区**: 输入要转换的文本
- **操作按钮**: 
  - 开始生成
  - 播放
  - 保存

### Voice Clone 界面

用于克隆参考音频的语音风格：

- **参考音频上传**: 上传参考音频文件
- **参考音频文本**: 输入参考音频的内容（可选）
- **文本输入区**: 输入要转换的文本
- **操作按钮**: 
  - 开始克隆
  - 播放
  - 另存为

### Audio Browser 界面

用于管理和播放生成的音频文件：

- **音频文件列表**: 显示生成的音频文件，按修改时间排序
- **操作按钮**: 
  - 打开文件夹
  - 刷新
- **音频播放器**: 播放选中的音频文件，显示播放进度和时间

### Settings 界面

用于配置软件的语言设置：

- **语言选择**: 选择界面语言（简体中文或 English）
- **关闭程序按钮**: 保存设置并关闭程序

### About 界面

显示软件的版本信息、开发者信息和许可证信息。

### Wizard 界面

首次运行时显示的环境设置向导：

- **安装类型选择**: 选择 GPU 版本或 CPU 版本
- **镜像源选择**: 选择阿里云镜像或官方源
- **安装进度**: 显示环境安装的进度和日志

## 界面操作技巧

### 音频浏览器操作

1. **打开音频浏览器**: 点击顶部导航栏的 "Audio Browser" 选项卡
2. **浏览音频文件**: 在文件列表中查看生成的音频文件
3. **播放音频**: 双击文件列表中的音频文件进行播放
4. **打开文件夹**: 点击「打开文件夹」按钮直接打开音频保存目录
5. **刷新列表**: 点击「刷新」按钮更新文件列表

### 设置操作

1. **打开设置**: 点击顶部导航栏的 "Settings" 选项卡
2. **更改语言**: 在语言下拉菜单中选择需要的语言
3. **保存设置**: 点击「关闭程序」按钮保存设置并关闭程序
4. **重启生效**: 重新启动软件，语言设置将生效

## 故障排除

### 界面显示问题

如果遇到界面显示问题，可尝试以下解决方法：

1. **更新显卡驱动**: 确保显卡驱动是最新的
2. **以兼容模式运行**: 右键软件图标→「属性」→「兼容性」→「以兼容模式运行」

### 界面响应缓慢

如果界面响应缓慢，可尝试以下解决方法：

1. **减少同时处理的任务**: 关闭其他占用系统资源的程序
2. **升级硬件**: 如果硬件配置较低，考虑升级 CPU 或内存或显卡
